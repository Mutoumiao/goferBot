# 知识库选择器实现计划

> **For agentic workers:** 必需子技能：superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans。步骤使用复选框（`- [ ]`）语法追踪。

**目标：** 实现多知识库选择 UI，支持在对话时通过 `@` 提及选择知识库，发送消息时携带 knowledgeBaseIds。

**架构：** 扩展现有 `ChatInput.vue` 的 `@` 触发逻辑，将 `KbMentionDropdown.vue` 升级为支持多选复选框的 `KbSelector.vue`。已选知识库状态由 ChatInput 组件本地管理，发送时通过事件传递给父组件。

**技术栈：** Vue 3 + Tailwind CSS + shadcn-vue

**Issue 引用：** [docs/02-issues/f-11-kb-selector.md](../../02-issues/f-11-kb-selector.md)
**Spec 引用：** [docs/03-specs/features/kb-selector/](../../03-specs/features/kb-selector/)

interface Props {
  knowledgeBases: KnowledgeBase[]
  selectedIds: string[]

}

  unselect: [kbId: string]

- [ ] **步骤 2: 实现键盘导航**

```typescript
const selectedIndex = ref(0)

function handleKeydown(e: KeyboardEvent) {
  if (!props.visible) return
  if (e.key === 'ArrowDown') {
    e.preventDefault()
    selectedIndex.value = (selectedIndex.value + 1) % props.knowledgeBases.length
  } else if (e.key === 'ArrowUp') {
    e.preventDefault()
    selectedIndex.value = (selectedIndex.value - 1 + props.knowledgeBases.length) % props.knowledgeBases.length
  } else if (e.key === 'Enter') {
    e.preventDefault()
    const kb = props.knowledgeBases[selectedIndex.value]
    if (kb) toggleKb(kb)
  } else if (e.key === 'Escape') {
    emit('close')
  }
}

defineExpose({ handleKeydown })
```

- [ ] **步骤 3: 实现多选复选框 UI**

```vue
<template>
  <div
    v-if="visible"
    class="absolute bottom-full left-0 mb-2 max-h-48 w-72 overflow-y-auto rounded-xl border border-border-default bg-white shadow-xl"
  >
    <div v-if="loading" class="p-3 space-y-2">
      <Skeleton v-for="i in 3" :key="i" class="h-8 w-full" />
    </div>
    <div v-else-if="knowledgeBases.length === 0" class="p-4 text-sm text-text-secondary text-center">
      请先创建知识库
    </div>
    <div
      v-for="(kb, i) in knowledgeBases"
      :key="kb.id"
      :class="[
        'flex cursor-pointer items-center gap-2 px-3 py-2 text-sm transition-colors',
        i === selectedIndex ? 'bg-accent-soft text-accent-500' : 'text-text-primary hover:bg-surface-2',
      ]"
      @mousedown.prevent="toggleKb(kb)"
    >
      <Checkbox :checked="selectedIds.includes(kb.id)" />
      <DatabaseIcon class="size-4 text-text-secondary" />
      <span class="truncate">{{ kb.name }}</span>
      <span class="ml-auto text-xs text-text-tertiary">{{ kb.documentCount || 0 }} 文档</span>
    </div>
  </div>
</template>
```
