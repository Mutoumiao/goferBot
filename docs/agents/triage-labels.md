# 分类标签

技能使用五个标准分类角色来交流。本文件将这些角色映射到本仓库 issue tracker 中实际使用的标签字符串。

| mattpocock/skills 中的标签 | 我们 tracker 中的标签 | 含义                           |
| -------------------------- | -------------------- | ------------------------------ |
| `needs-triage`             | `needs-triage`       | 维护者需要评估此问题           |
| `needs-info`               | `needs-info`         | 等待报告者补充更多信息         |
| `ready-for-agent`          | `ready-for-agent`    | 已完全明确，可由 AFK Agent 处理 |
| `ready-for-human`          | `ready-for-human`    | 需要人工实现                   |
| `wontfix`                  | `wontfix`            | 不处理                         |

当技能提到某个角色（例如"应用 AFK-ready 分类标签"）时，请使用本表中对应的标签字符串。

如需匹配你实际使用的词汇，请编辑右侧列。
