# 行为规格：知识库页面迁移

> 状态：draft | 关联 issue：f-36

---

## 1. KB 列表交互

| 事件 | 行为 |
|------|------|
| 页面加载 | `useRequest` 自动 GET /kb，显示 loading 骨架屏 |
| 加载成功 | 渲染 KB 卡片列表，显示名称+文档数+日期 |
| 列表为空 | 显示"暂无知识库" + "创建"按钮 |
| 加载失败 | 显示错误信息 + 重试按钮 |
| 点击"创建知识库" | `openDialog(CreateKbDialog)` |

## 2. KB 创建 Dialog

| 事件 | 行为 |
|------|------|
| Dialog 打开 | 显示空表单：名称输入框 + 描述 textarea |
| 点击确认 | `useForm.send()` → POST /kb |
| 创建中 | 按钮 disabled + spinner |
| 创建成功 | 关闭 Dialog → 刷新列表 |
| 创建失败 | 显示错误提示，Dialog 保持打开 |

## 3. 文件上传

| 事件 | 行为 |
|------|------|
| 拖拽/选择文件 | 触发 `useUploader`，显示进度条 |
| 上传中 | 进度百分比实时更新 |
| 上传完成 | 文件出现在文档列表中 |

## 4. 错误场景

| 场景 | 用户可见 |
|------|----------|
| KB 名称重复（409） | "知识库名称已存在" |
| 网络错误 | "网络错误，请重试" |
| 文件过大 | "文件大小超过限制" |
