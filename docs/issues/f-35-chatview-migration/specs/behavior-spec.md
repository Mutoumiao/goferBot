# 行为规格：ChatView 页面迁移

> 状态：draft | 关联 issue：f-35

---

## 1. 消息发送交互

| 事件 | 行为 | 后置状态 |
|------|------|----------|
| 输入框为空 → 点击发送 | 按钮 disabled | idle |
| 输入框有内容 → 点击发送/Enter | 发送用户消息 → 显示占位"思考中..." → 建立 SSE 连接 | streaming |
| SSE 接收 token | 逐字追加到 AI 回复气泡 | streaming |
| SSE 流结束 | 标记消息完成，移除"思考中" | idle |
| SSE 连接异常 | 显示"连接中断" + 重试按钮 | error |
| 发送中再次点击发送 | 忽略（按钮 disabled） | streaming |

## 2. 历史记录

| 事件 | 行为 |
|------|------|
| 进入页面 | 加载最近 20 条消息 |
| 滚动到顶部 | 触发 `loadMore()`，加载更早消息 |
| 加载历史中 | 顶部显示 spinner |
| 全部加载完 | 显示"已加载全部消息" |

## 3. 错误场景

| 场景 | 用户可见 |
|------|----------|
| 网络断开 | 消息列表最后一个 AI 消息显示"⚠ 发送失败" + 重试按钮 |
| SSR 流中断 | "连接中断，点击重试" |
| 历史加载失败 | 顶部显示"加载失败，点击重试" |
