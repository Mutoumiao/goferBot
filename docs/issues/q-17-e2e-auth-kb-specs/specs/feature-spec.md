# q-17 功能规格

## 概述

认证流程和知识库生命周期的端到端测试，验证用户从注册到管理知识库的完整体验。

## 功能需求

### FR-01: 认证流程测试
- 注册页面表单验证
- RSA 密码加密传输
- 注册成功自动登录
- 登录成功 JWT 存储
- Token 刷新机制
- 路由守卫行为

### FR-02: 知识库生命周期测试
- 知识库列表加载
- 创建新知识库
- 进入知识库详情
- 上传文档（txt/md/pdf）
- 删除知识库（软删除）
- 回收站恢复
