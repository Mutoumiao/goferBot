## Frontmatter 字段说明

| 字段 | 说明 | 必填 |
|------|------|------|
| `issue_id` | 对应 issue 编号 | ✅ |
| `type` | 固定值：`plan` | ✅ |
| `version` | 版本号 N（v{N}.md 中的 N） | ✅ |
| `status` | draft → ready → executing → done | ✅ |
| `task_count` | 任务总数 | 可选 |
| `summary` | 清晰描述技术方案与任务结构，Agent 据此判断是否需深入阅读 | ✅ |

## 正文填写说明

| 字段 | 说明 |
|------|------|
| `目标` | 一句话概括计划要交付什么 |
| `任务` | 每个任务不超过 8 个步骤 |
| `步骤` | 必须包含具体代码示例和验证命令 |
| `规格覆盖检查` | 自检是否覆盖 spec 中所有要求 |
