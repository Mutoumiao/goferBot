# V1 架构废弃代码清理 实现计划

> **For agentic workers:** 必需子技能：`superpowers:subagent-driven-development`（推荐）或 `superpowers:executing-plans`。步骤使用复选框（`- [ ]`）语法追踪。

**目标：** 清理 V1 架构遗留的废弃代码（SQLite 本地存储、物理文件夹同步、sqlite-vec 索引），减少技术债务，避免后续开发者误用旧机制。

**架构：** 按依赖拓扑顺序从叶子到根逐步删除/清空文件，先切断内部依赖再清理路由，最后移除 package.json 依赖并验证编译与启动。所有路由保留 Hono 框架并返回 501。

**技术栈：** TypeScript + Hono + pnpm

**Issue 引用：** [docs/issues/q-03-v1-cleanup.md](../../../issues/q-03-v1-cleanup.md)
**Spec 引用：** [docs/03-specs/q-03-v1-cleanup/](../../../03-specs/q-03-v1-cleanup/)
**测试用例：** 无新增测试用例（纯清理任务，验证现有命令通过即可）
