# 行为规格：知识库文件管理器

> 对应 issue: `f-06-knowledge-base-file-manager`

---

## 入口

- 路由：`/knowledge-bases`
- 触发：用户在左侧知识库列表中点击某个知识库，右侧显示该知识库的文件管理器

## 初始状态

- 若未选中知识库：显示占位提示"请选择一个知识库"
- 若选中知识库：加载该知识库根目录下的文件和文件夹

## 交互状态

| 状态 | 视觉 | 用户操作 | 系统响应 |
|------|------|----------|----------|
| loading | 骨架屏（6 个灰色方块） | 禁用 | 发送 API 请求加载文件列表 |
| empty | 文件夹图标 + "点击添加文件导入文档" | 点击上传按钮 | 打开文件选择对话框 |
| error | 错误提示 + 重试按钮 | 点击重试 | 重新加载 |
| success | 图标网格显示文件和文件夹 | 双击/右键/多选 | 对应操作 |
| partial | 部分文件加载成功，部分失败 | 重试失败项 | 重新请求 |

## 正常流程

| 步骤 | 用户操作 | 系统响应 | 视觉状态 |
|------|----------|----------|----------|
| 1 | 点击知识库列表项 | 右侧加载文件管理器 | loading |
| 2 | 数据返回 | 显示文件/文件夹网格 | success |
| 3 | 双击文件夹 | 进入该文件夹，面包屑更新 | loading → success |
| 4 | 点击面包屑父级 | 返回上级目录 | success |

## 错误场景

| 场景 | 触发 | 视觉 | 恢复 |
|------|------|------|------|
| 加载失败 | API 500/网络错误 | 红色提示条 + 重试按钮 | 点击重试 |
| 无权访问 | 403 | "无权访问该知识库" | 返回列表 |
| 文件夹不存在 | 404 | "文件夹不存在" | 返回根目录 |

## 动画与过渡

| 过渡 | 持续时间 | 缓动 | 备注 |
|------|----------|------|------|
| 文件夹进入 | 200ms | ease-out | 内容淡入 |
| 面包屑切换 | 150ms | ease-in-out | 路径项滑动 |
| 选中项 | 100ms | ease | 边框高亮 |

## 无障碍

- 键盘导航：Tab 切换文件项，Enter 打开，Backspace 返回上级
- 屏幕阅读器：文件项 aria-label 包含"文件名，类型，状态"
- 触摸目标：文件项最小 44x44px

## 响应式行为

| 断点 | 布局变化 |
|------|----------|
| 375px | 2 列网格 |
| 768px | 3 列网格 |
| 1024px | 4 列网格 |
| 1440px | 5 列网格 |
