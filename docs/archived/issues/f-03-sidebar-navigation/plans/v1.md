# Sidebar 导航剩余工作实现计划

> **For agentic workers:** 必需子技能：superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans。步骤使用复选框（`- [ ]`）语法追踪。

**目标：** 完成 issue `f-03-sidebar-navigation` 的剩余工作：Tooltip 悬停提示、移动端底部导航适配、组件迁移到 `layout/` 目录、登录/注册页隐藏 Sidebar。

**架构：** 将现有 `SideBar.vue` 迁移并重命名为 `AppSidebar.vue`，置于 `components/layout/` 目录。桌面端保持 64px 左侧边栏（上下分区），移动端（< 768px）切换为 48px 底部导航栏。使用 shadcn-vue Tooltip（reka-ui）实现悬停 300ms 延迟提示。通过路由元信息 `meta.hideSidebar` 控制登录/注册页不渲染 Sidebar。

**技术栈：** Vue 3 + TypeScript + Tailwind CSS v4 + vue-router + shadcn-vue Tooltip (reka-ui) + lucide-vue-next

- [ ] **步骤 1: 替换 `SideBar` 为 `AppSidebar`，添加 `hideSidebar` 计算属性**

  将 `packages/webui/src/App.vue` 完整替换为：

  ```vue
  <script setup lang="ts">
  import { computed, onMounted } from 'vue'
  import { useRoute, useRouter } from 'vue-router'
  import AppSidebar from './components/layout/AppSidebar.vue'
  import { useSettingsStore } from './stores/settings'

  const route = useRoute()
  const router = useRouter()
  const settingsStore = useSettingsStore()

  const hideSidebar = computed(() => {
    return route.meta.hideSidebar === true
  })

  onMounted(async () => {
    settingsStore.loadConfig()
  })

  function handleNavigate(name: string) {
    router.push({ name })
  }
  </script>

  <template>
    <div class="flex h-screen bg-surface-1 text-text-primary">
      <AppSidebar
        v-if="!hideSidebar"
        :active-type="(route.name as string) ?? 'chat'"
        @navigate="handleNavigate"
      />
      <main
        :class="[
          'flex-1 overflow-hidden',
          !hideSidebar ? 'pb-12 md:pb-0' : '',
        ]"
      >
        <RouterView />
      </main>
    </div>
  </template>
  ```

  说明：
  - `hideSidebar` 计算属性读取当前路由的 `meta.hideSidebar`
  - `v-if="!hideSidebar"` 控制 Sidebar 渲染
  - `main` 元素在移动端（有 Sidebar 时）添加 `pb-12`，为底部 48px 导航栏留出空间；桌面端 `md:pb-0` 恢复无内边距
  - 单一 `navigate` emit 替代原有的 5 个独立 emit
