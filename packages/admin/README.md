# GoferBot Admin

Admin management console for the GoferBot platform.

## Tech Stack
- React 19 + TanStack Start
- Ant Design v5 + @ant-design/pro-components
- alova v3 (HTTP client) + ahooks
- Zustand (state management)
- Tailwind CSS v4

## Development
```bash
pnpm --filter @goferbot/admin dev
```

Runs on port **1421**.
